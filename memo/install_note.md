## インストール
【 Ubuntu 16 】
```
sudo apt-get update 
sudo apt-get upgrade

sudo apt-get install nodejs
sudo apt-get install npm
sudo npm install -g n
sudo n latest
sudo npm update -g npm
sudo npm update -g
sudo npm outdated -g
sudo npm install -g express
sudo npm install -g express-generator
```

## バージョン確認
```
node -v
npm -v
```
